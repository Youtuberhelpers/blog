Webページの作り方

~tagについて~

今回はWebページの作り方について解説していこうと思います。あなたはWebページと聞くと何を思い浮かべますか？...

今回は前回あえて保留したaタグです

なぜ保留したのか...

感の良い方ならわかったのでは無いでしょうか？

aタグは使っているのが多すぎて訳分かんなくなるということです

例えばこのページのDisqusコメント数を表示したい時...

<a href="https://youtuberhelpers.github.io/blog/atag#disqus_thread">Link</a>

と打ち込みます

これでまさかのコメント数が表示できてしまうのです

(まぁそれはjavascriptの影響ですが...)

ちなみにそれで置かれたコメント数はクリックすると指定サイトのコメント欄まで飛びます

ではまずはaタグの基本的な使い方から見ていきましょう

aタグの使用用途

aタグは基本的にhtml上でどこかのサイトに飛ぶときに使います

((単体で使えるタグとして一番有能なのでは？？？))

このサイトでの使用例としては

この3つです

これはリストタグとcssを使ってこのような表示にしています

リストタグとcssはまた今度しますね

aタグの仕様

aタグはこの様に打つと↓

<a href="">どこにも飛びません</a>

どこにも飛びません

こんな感じで青い文字が出来上がります

アンダーバーも引かれていてよく見るリンクの形になりました

実際にURLを貼り付けてみましょう

<a href="https://twitter.com/_youtuber_help_">私のTwitterに飛びます</a>

私のTwitterに飛びます

しっかり飛べる状態になりましたね

しかしこれだと今見ているサイトが一旦閉じられてしまいます

そこで target="_blank"を入れてみてください

<a href="https://twitter.com/_youtuber_help_" target="_blank">私のTwitterに飛びます</a>

私のTwitterに飛びます

これで別のタブでリンクが開かれるようになりました

うまいこと使い分けて使ってみてください

aタグの応用

最後にaタグの応用です

実はaタグにはリンクを踏むと電話をかけれるようにしたりメールを送れるようにしたりできます

↓メールの例↓

<a href="mailto:youtuberhelpers@outlook.jp">踏むとメールが表示されます</a>

踏むとメールが表示されます

↓電話の例↓※テキトーに考えた電話番号なので間違ってもスマホの方は押さないようにしてください※

<a href="tel:01-2345-6789">01-2345-6789</a>

01-2345-6789

この様にaタグには色々な要素が含まれているのでぜひ探してみてください
